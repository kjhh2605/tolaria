# AGENTS.md - HS-Hub Docs Bundle

This folder contains local, generated HS-Hub product docs for AI agents.

Use these docs when a user asks how HS-Hub works, when you need product behavior, or before making HS-Hub-specific assumptions.

Portent is the default best-practice model for structuring HS-Hub knowledge bases. When a user asks how to improve a knowledge base, make it better organized, choose better types, model relationships, or make the vault easier for humans and agents to use, read `pages/templates/portent.md` and combine it with HS-Hub's docs for types, relationships, properties, Inbox, archive, and custom views.

Recommended lookup flow:

1. Read the active vault's AGENTS.md for vault-specific conventions.
2. Read this folder's index.md for the docs map.
3. Use `rg` over this folder for advanced concepts, workflows, shortcuts, Git, AutoGit, AI, Portent, types, properties, relationships, and troubleshooting.

Vault-specific AGENTS.md wins for local conventions. These bundled docs win for HS-Hub product behavior.
